### Prerequisites

1. Clone the repo:
    ```sh
    git@github.com:ALTA-PROJECT2-GROUP4/DashboardQ-FE.git
     ```
2. Install dependencies using npm:
    ```sh
   cd name-your-project
   ```
3. install module use:
    ```sh
    npm i
    ```
4. Start the development server:
    ```sh
    npm run dev / npm start
    ```
